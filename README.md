# Projet JEE – Architecture Microservices avec Spring Cloud

## Description
Ce projet s’inscrit dans le cadre du module JEE.  
Il consiste à mettre en œuvre une architecture microservices basée sur Spring Boot et Spring Cloud.

Le projet est composé de deux études de cas :
- Étude de cas 1 : Microservice Commandes avec configuration centralisée
- Étude de cas 2 : Architecture microservices complète (à venir)

---

## Technologies utilisées
- Java 17
- Spring Boot
- Spring Data JPA
- Spring Cloud Config
- Spring Boot Actuator
- H2 Database
- Maven

---

## Organisation du dépôt
- `cas-1-microservice-commandes` : Étude de cas 1 (terminée)
- `cas-2-microservices` : Étude de cas 2 (en cours)
